## 1. Concept Definition

**Family Functioning** 指家庭作为系统，在处理日常任务、满足成员需求、维持关系健康方面的整体运作能力。它关注的是家庭内部的结构、组织特征与成员互动模式。

Family Functioning 是一个**系统层面**的构念，不等同于单一成员的功能状态，而是家庭作为整体如何运作。

---

## 2. Theoretical Frameworks

Family Functioning 有两个主要的理论框架，各有不同的测量工具：

### 框架 A：McMaster Model of Family Functioning

**工具：** Family Assessment Device (FAD)
**Repository 锚定框架**（ET-000033 canonical reference）

| 维度 | 内容 |
|---|---|
| Problem Solving | 家庭识别和解决问题的能力 |
| Communication | 信息在家庭成员间的交换方式 |
| Roles | 家庭成员各自承担的职能分工 |
| Affective Responsiveness | 对情感刺激的适当反应能力 |
| Affective Involvement | 家庭成员彼此情感投入的程度 |
| Behavior Control | 家庭管理行为的规则与模式 |
| General Functioning | 整体家庭健康与功能的综合指标 |

### 框架 B：Circumplex Model（Olson）

**工具：** Family Adaptability and Cohesion Scale (FACES)
**在 ET-000036 中贡献证据**

| 维度 | 内容 |
|---|---|
| Cohesion | 家庭成员之间的情感联结程度 |
| Adaptability / Flexibility | 家庭应对变化和压力的灵活程度 |
| Communication | 家庭沟通质量（促进因素） |

---

## 3. Framework Equivalence Note

两个框架均属于 Family Functioning 概念，但关注维度不同：

```
Family Functioning (Concept)
├── McMaster Model / FAD    → ET-000033 (canonical in Repository)
└── Circumplex Model / FACES → ET-000036 (contributing evidence)

Framework Equivalence Principle:
Both contribute to the same Concept.
Framework differences must be documented.
Do NOT treat them as interchangeable tools.
```

在跨研究比较或临床解释时，需确认所用工具属于哪个框架。

## 4. Clinical Relevance

- 家庭功能状态影响家庭成员的心理健康、亲子关系质量和儿童发展
- 在障碍儿童家庭中，Family Functioning 的评估有助于识别家庭系统层面的支持需求
- FAD 可用于筛查家庭功能困难，指导家庭导向的干预

## 6. Known Conceptual Boundaries

- Family Functioning ≠ Family Support（前者是内部运作能力，后者是外部资源获取）
- Family Functioning ≠ Parenting Stress（前者是系统属性，后者是个体心理状态）
- 家庭人口学变量（收入、教育程度、家庭类型）**不等同于** Family Functioning 的操作化测量
- 两个框架（McMaster / Circumplex）不可相互替代
