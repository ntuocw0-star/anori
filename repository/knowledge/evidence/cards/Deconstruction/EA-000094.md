## Layer 0 · 为什么值得读

> 它把一个抽象的问题变得具体：**家长在日常里，实际上在用什么方式帮助孩子参与家庭和社区活动？**

EA-000092 告诉我们环境解释了一半以上的参与差异，EA-000093 告诉我们具体有哪些环境因素。这篇研究做了下一步：它访问了 ASD 儿童的家长，问他们实际在做什么——面对这些环境因素，家长用了哪些策略来创造参与机会。

这直接连接了两个 Topic：Family Support（家长如何保持自己的可持续性）和 Participation（家长如何帮助孩子真正参与）。家长不只是被动地受环境影响——他们也是主动修改参与条件的人。

对于 Anori Repository：这是 Cluster 4（Family Participation Support）的核心证据来源，也是整个 Participation Topic 和 Family Support Topic 之间最重要的连接节点。

## Layer 2 · Key Findings

### F1：家长策略的主要类型

家长报告了多种支持 ASD 儿童参与的策略，归纳为以下几类：

**策略一：提前准备和环境预告（Preparation and Priming）**

```
策略：在参与前，让孩子了解将要发生的事情
包括：
├── 社交故事 / 视觉支持
├── 提前访问活动场所（熟悉环境）
├── 与孩子讨论活动规则和顺序
└── 让孩子有选择权（去还是不去）

作用：
├── 降低活动的不可预测性 → 焦虑↓
└── 提高孩子的参与意愿
```

**策略二：活动选择和修改（Activity Selection and Modification）**

```
策略：选择适合孩子参与方式的活动，或修改现有活动
包括：
├── 选择低社交需求的活动
├── 优先选择孩子有特殊兴趣的活动类型
├── 要求组织者对活动规则进行调整
└── 在高社交需求活动中主动提供结构化支持

作用：
├── 使参与条件与孩子匹配
└── 减少参与障碍的直接来源
```

**策略三：陪同和逐步退出（Graduated Support）**

```
策略：家长陪同参与，并随时间逐步减少直接支持
包括：
├── 初期全程陪同（safety net）
├── 随孩子熟悉程度逐步后退
└── 在远处观察而非直接介入

作用：
├── 让孩子有安全的参与基础
└── 随时间积累独立参与的经验
```

**策略四：倡导（Advocacy）**

```
策略：家长主动向学校、社区、服务提供者争取参与支持
包括：
├── 向活动组织者解释孩子的 ASD 特征和需求
├── 要求服务系统提供调适
└── 在社区中建立对孩子参与方式的理解

作用：
├── 修改外部环境障碍
└── 建立更接受 ASD 参与方式的情境
```

---

### F2：环境因素与家长策略的互动

家长的策略，不是在真空里发挥作用的——它们是对环境因素的主动回应：

```
环境障碍（如社交需求高）
        ↓ 家长回应
→ 选择低社交需求活动
→ 提前准备
→ 活动修改请求

环境促进（如接受态度的环境）
        ↓ 家长利用
→ 减少倡导负担
→ 更多参与机会
→ 孩子参与意愿↑
```

---

### F3：家庭日常惯例与参与

研究发现，家庭的日常惯例（Routines）是参与机会的重要来源：

- 固定的家庭日常（吃饭、购物、散步）提供可预测的参与情境
- 日常惯例比特殊活动对参与的支持更稳定
- 家庭惯例的调整，可以是增加参与机会的低成本策略

### F5：本文没有建立的结论

- ❌ 没有量化策略效果（定性研究；没有干预组对照）
- ❌ 没有覆盖学校情境（→ EA-000095）
- ❌ 没有评估不同策略的相对效果
- ⚠️ 家长报告；可能有社会期望偏差
- ⚠️ 定性/混合方法；不能推断群体效应量

## Layer 3 · Parent Meaning

**很多家长后来才发现……**

**1. 他们一直在做的事情，研究里有名字——这些策略是有用的。**
很多家长后来才发现，研究记录了家长用来帮助孩子参与的方式：提前告诉孩子会发生什么、选择适合他的活动类型、陪他去但慢慢退后、向活动组织者解释孩子的需求。这些不是"随机摸索"，而是有证据记录的有效策略。

**2. 告诉别人孩子的需求，也是参与支持的一部分。**
很多家长后来才发现，研究把"倡导"（向外解释和争取支持）列为家长策略的重要类型。这不是"麻烦别人"，而是直接减少参与障碍——让周围的环境对孩子更接受，才能让更多参与成为可能。

**3. 日常的家庭惯例，本身就是参与机会。**
很多家长后来才发现，研究发现固定的家庭日常（吃饭、购物、散步）提供了可预测的参与情境——不需要专门安排，日常惯例本身就是孩子练习参与的空间。

## Layer 4 · Professional Meaning

**给 OT、早期干预工作者、家庭支持专业人员的启示：**

- **把家长策略纳入参与评估和计划**：家长是参与支持的核心实施者，不只是信息接收者——参与支持计划应识别家长现有的策略，并在此基础上扩展
- **Coaching 模型最适合支持家长策略**：帮助家长识别、反思和精进他们已经在用的策略，比直接指令更有效（与 EA-000071 FCC 框架一致）
- **识别和利用现有家庭惯例**：在参与支持中，不要只想到增加新的活动——先看现有的日常惯例里有哪些参与机会可以被支持和扩展
- **倡导技能是可以培训的**：帮助家长知道如何向学校、社区和服务提供者有效倡导，是减少环境障碍的间接方式

## ET 素材池

| 研究语言 | ET 方向（Anori Voice）|
|----------|----------------------|
| Parent strategies documented | 研究记录了家长实际在用的参与支持策略——提前准备、选择合适的活动、陪同然后慢慢退后、向别人解释孩子的需求 |
| Advocacy is strategy | 向活动组织者或学校解释孩子的参与方式，是一种直接减少障碍的策略——不是要求特殊照顾，而是让环境更适合孩子参与 |
| Routines as participation | 日常惯例（吃饭、购物、散步）本身就是参与机会——不需要专门安排，先看看现有的日常里有什么 |
| Graduated support | 陪孩子去，然后慢慢后退——这不是依赖，而是建立参与经验的方式 |
| Connect to Family Support | 家长自己的状态和资源，影响他们能用多少精力来支持孩子的参与——两件事是连在一起的 |
