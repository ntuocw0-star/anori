## Layer 0 · 为什么值得读

> 它不是问"家长培训有没有用"——关于降低压力和改善儿童行为的证据已经很多了。它问的是一个更精准的问题：**家长培训能不能提高家长对自己照顾能力的信心本身？**

这是一个重要区分。很多研究发现家长培训改善了儿童行为或降低了家长压力，但那不等于家长的效能感信念改变了。这篇综述专门找那些把 PSE 作为主要结果变量、独立测量的研究，然后回答：是的，可以——但效果的大小和稳定性因干预类型而异。

效应量 SMD=0.60 是一个中等偏强的效应，但 I²=74% 的高异质性说明"亲职培训"不是一个统一的东西——不同类型的培训对 PSE 的效果差异很大。

对于 Anori Repository：这是 Parent Self-efficacy Cluster 的收束节点，回答"什么干预能改善 ASD 家长的效能感信念"。它把 Bandura 的四个效能感来源（EA-000072）和实证干预证据连接起来，形成可以指导实践的框架。

## Layer 1 · Research Question

论文真正回答的问题是：

> **亲职干预项目能否有效提高 NDD（含 ASD）儿童家长的 PSE？效果大小如何？哪些因素调节了干预效果？**

## Layer 2 · Key Findings

### I1：检索与纳入

- **检索策略**：系统检索（数据库包括 PsycINFO、PubMed、ERIC 等）
- **纳入标准**：前瞻性研究；NDD 儿童家长；亲职干预；PSE 为主要或次要结果变量；量化 PSE 测量
- **纳入研究数**：**25 项研究**
- **样本特征**：混合 NDD（包含 ASD、Down syndrome、ADHD 等）；儿童年龄 0–18 岁

---

### I2：总体效应量

```
SMD = 0.60（95% CI: 0.38–0.83）
解读：中等偏强效应（Cohen's d ≈ 0.6 对应中等效应）
含义：亲职培训后，家长 PSE 平均提升约 0.6 个标准差

I² = 74%（高异质性）
含义：不同研究的效果差异大，不应把 SMD=0.60 视为所有培训的一致预期效果
每项研究自身的结果更有参考价值，汇总效应是下限估计
```

**结论：亲职培训对 NDD 家长 PSE 有统计显著的中等效应，但异质性高，提示效果因干预类型和对象而异。**

---

### I3：调节变量——儿童年龄是最重要的调节因素

> **<5 岁儿童家长的 PSE 提升效果最大。**

```
<5 岁儿童家长
→ 亲职技能和效能感仍在建立中
→ 对成功经验的反应更敏感
→ 干预效果最强

学龄期及以上
→ 亲职模式可能已固化
→ 效能感改变需要更多成功经验积累
→ 干预效果相对较小但仍显著
```

**对 ASD 家庭的含义：** 诊断后早期（通常为 2–5 岁）是 PSE 干预效果最好的窗口，这与 Dai et al.（2024）关于诊断后第一年是关键窗口的纵向发现高度一致。

---

### I4：按 Bandura 四来源分类的干预机制

本综述的干预可以按 Bandura 四来源重新分类：

| 效能来源 | 对应干预方式 | PSE 效果 |
|---|---|---|
| **Mastery Experience（绩效成就）** | 技能培训 + 现场实践 + 即时反馈 | ★★★★★ 最强；直接产生成功经验 |
| **Vicarious Experience（替代性经验）** | 录像示范 + 同伴家长小组 | ★★★★☆ 强；观察他人成功 → 效能感↑ |
| **Verbal Persuasion（言语说服）** | Coaching + 正向反馈 + 优势导向对话 | ★★★★☆ 强；当与 Mastery 结合时最有效 |
| **Physiological Regulation（生理调节）** | 正念培训 + 压力管理 | ★★★☆☆ 中等；PSE 为次要结果 |

**关键发现：最有效的干预同时包含 Mastery Experience（实践成功）和 Verbal Persuasion（正向反馈），而不只是知识传授。**

---

### I5：各干预模式的证据

| 干预模式 | PSE 效果 | 说明 |
|---|---|---|
| **Parent-mediated NDBI（自然发展行为干预）** | ↑↑ PSE | 家长学习特定技能并执行 → 成功经验 → 效能感↑；学龄前效果最显著 |
| **Triple P / Stepping Stones Triple P** | ↑ PSE（RCT 证据）| 正面亲职计划；结构化技能培训 + 即时反馈 |
| **Coaching Model（现场指导）** | ↑↑ PSE | 专门针对效能感的反馈机制；Verbal Persuasion 最直接 |
| **Group Parent Training（团体培训）** | ↑ PSE | 同时提供 Vicarious Experience（观察他人）和 Verbal Persuasion |
| **Mindfulness-based Intervention** | △ PSE | PSE 为次要结果；主要改善压力；对效能感的独立效应有限 |
| **Psychoeducation（心理教育）** | △ PSE | 知识增加不自动提升效能感信念；需要结合实践机会 |

---

### I6：关键边界——技能 vs. 效能感

这是使用本 EA 最需要注意的边界：

```
Skill Improvement（技能提升）
≠
Self-efficacy Improvement（效能感提升）

干预可能：
→ 提高技能但不改变效能感（知道但不相信自己能做到）
→ 提高效能感但技能未跟上（过高的自信）
→ 同时改善两者（最佳干预效果）

评估干预对 PSE 的效果，必须：
→ 独立测量 PSE（不只是技能测试或儿童行为观察）
→ PSE 的测量工具需要区分 Efficacy 和 Satisfaction
→ 避免用压力下降替代 PSE 改善（两者不等同）
```

---

### I7：本文没有建立的结论

- ❌ 没有建立 ASD-specific 亲职培训对 PSE 的独立效应量
- ❌ 没有建立 PSE 改善 → 压力降低的中介路径（需要同时测量两者并检验关联）
- ❌ 没有评估干预效果的长期维持（随访数据有限）
- ❌ 没有比较不同干预类型的相对效果（无网络 Meta 分析）
- ⚠️ 混合 NDD 样本；ASD-specific 效应量无法直接从本综述提取
- ⚠️ I²=74% 高异质性——SMD=0.60 是估计均值，不是稳定预期值
- ⚠️ 2018 年以来 ASD PSE 干预研究快速增加；本综述不覆盖最新证据

## Layer 3 · Parent Meaning

**很多家长后来才发现……**

**1. 参加家长培训，不只是学技能——研究发现它也能改变你对自己的信心。**
很多家长后来才发现，研究者把 25 项专门测量家长信心的研究加在一起，发现：参加家长培训后，家长对自己照顾能力的信心平均提升了约 0.6 个标准差。这不是说"你会有一点进步"，而是一个在统计上中等偏强的效应。

**2. 越早参加，效果越好——尤其是孩子 5 岁前。**
很多家长后来才发现，综述发现孩子越小，家长参加培训后效能感提升越大。这不是说晚了就没有用，而是说早期是最值得投入的时间窗口，那时候信心还在建立中，对成功经验的反应最敏感。

**3. 让你真正实践并得到具体反馈的培训，比只是告诉你知识的培训效果更好。**
很多家长后来才发现，研究发现：光是听讲座、学知识，对效能感的改变有限。真正有效的是让你自己动手做，并且有人在旁边告诉你"你刚才做对了"。这是两件截然不同的体验。

**4. 技能学会了，不等于信心就来了——这两件事需要分开支持。**
很多家长后来才发现，研究特别指出：技能提升和效能感提升不是同一件事。你可能学会了一个技术，但还是不相信自己在真实情况下能做到。这时候需要的不是更多知识，而是更多真实的成功机会。

## Layer 4 · Professional Meaning

**给语言治疗师、特教老师、OT、儿科医生、心理咨询师的启示：**

- **在家长培训中，明确设计"产生效能感"的成分**：不只是教技能，同时设计可以成功的小任务、提供即时正向具体反馈、邀请同伴家长分享
- **评估培训效果时，使用 PSE 量表**（如 PSOC Efficacy 子量表），而不只是儿童行为改变或家长压力——两者可能分离
- **早期干预期（0–5 岁）是 PSE 干预效果最大的窗口**：在早期干预项目中，主动加入 PSE 增强成分
- **Coaching（现场指导 + 正向反馈）是最高效的单一 PSE 提升策略**：在接触家长时，每次包含一个明确的正向、具体的反馈——"你刚才在他哭的时候等了 5 秒再回应，那是对的"
- ⚠️ **I²=74% 提醒**：不同培训项目效果差异很大，不能假设任何培训都会提高 PSE——需要在当地情境中验证

## Layer 5 · Actionable Implications

### 今天可以做什么

**对家长：**
- 在选择家长培训时，优先选择包含**实践机会 + 现场反馈**的项目，而不只是讲座或手册——那些让你真正做了并且有人告诉你做对了的培训，对信心的改变更大
- 如果参加培训后感到"我知道了但还是没有信心"，这是正常的——知识和信心是两回事，需要更多真实的成功经验来填补这个空缺

**对服务提供者：**
- 在设计家长支持项目时，检查以下四个问题：这个项目有没有让家长真正实践？有没有立即告诉家长他们做对了什么？有没有让家长看到其他家长成功？有没有帮助家长管理疲劳和情绪？
- 如果项目只有知识传授，不要期望它会显著改变 PSE
