## Layer 0 · 为什么值得读

> 它回答了一个让整个参与研究领域都面红的问题：**我们说了这么多年"测量参与"，但我们到底在测什么？**

fPRC 框架（EA-000083）清楚地区分了 Attendance（出席）和 Involvement（投入），并强调 Involvement 才是参与质量的核心。但如果现有研究使用的测量工具根本没有测量 Involvement，那么大量声称"改善了参与"的干预研究，实际上只证明了"孩子出席了更多次"，而不是"孩子参与得更好了"。

Adair et al.（2018）系统梳理了 118 个被用作参与测量的工具，用 fPRC 的五个构念框架对它们逐一映射。结论令人警醒：真正测量 Involvement 的工具极少，绝大多数工具测量的是 Attendance，许多工具甚至测量的是活动能力，而不是参与本身。

对于 Anori Repository：这篇综述提供了 Participation Topic 的测量全景和关键缺口，是 Cluster 2（ASD 参与画像）证据评估和 Cluster 5（参与干预）效果评估的方法论基础——每次声称"改善参与"，都需要问：用的是什么工具，测量的是哪个维度？

## Layer 1 · Research Question

论文真正回答的问题是：

> **儿童障碍研究中使用的参与测量工具，实际上测量了 fPRC 框架中的哪些构念？哪些构念被充分测量，哪些存在缺口？**

## Layer 2 · Key Findings

### O1：检索与纳入

- **数据库检索**：PubMed、PsycINFO、CINAHL、Embase 等
- **初始记录**：32,767 条
- **最终纳入研究**：578 篇
- **识别工具数**：118 个不同工具被用于测量儿童参与
- **进入 fPRC 构念映射**：51 个工具（有足够信息进行映射分析）

---

### O2：最关键的发现——Involvement 测量极度不足

```
118个工具的 fPRC 构念覆盖分布：

Attendance（出席/频率）
→ 21 个工具明确测量
→ 是被测量最多的单一构念

Involvement（投入程度）
→ 仅 10 个工具将 Involvement 作为独立量表测量
→ 是所有构念中最被忽视的

Activity Competence（活动能力）
→ 大量工具涉及，但往往被误标为"参与"
→ 这正是"参与测量"概念混乱的主要来源

Preferences（偏好）
→ 少数工具涉及；CAPE 是最主要的来源

Sense of Self（自我感）
→ 极少工具涉及
```

**最重要的单一结论：**

> **绝大多数声称测量"参与"的工具，实际上测量的是出席频率（Attendance），而不是参与时的投入程度（Involvement）。**

---

### O3：常见误区——活动能力被误认为参与

```
错误等式（研究中常见）：
Activity Performance（活动表现）= Participation（参与）

正确区分：
Activity Performance
= 孩子能不能做一件事，做得怎样
= Activity Competence 的测量

Participation（Attendance + Involvement）
= 孩子实际在日常生活中去了哪里，投入了多少
= 需要专属的参与测量工具
```

这意味着：很多干预研究的结果（"改善了功能"或"改善了活动能力"）被错误地解读为"改善了参与"，因为使用的工具实际上是活动能力测量工具。

---

### O4：PEM-CY 和 CAPE 的定位

在 51 个进入构念映射的工具中：

**PEM-CY：**
- 同时覆盖 Attendance（Frequency 子量表）和 Involvement（Involvement 子量表）
- 是少数同时测量两个核心维度的工具之一
- 额外测量环境因素（Environment）
- 覆盖三个情境（家庭/学校/社区）

**CAPE（Children's Assessment of Participation and Enjoyment）：**
- 主要测量 Attendance（频率 + 多样性）
- 包含享受度（Enjoyment），接近 Involvement 的情感维度
- 但 Involvement 不作为独立量表
- 聚焦于休闲活动；不覆盖社区和学校结构性参与

---

### O5：测量缺口总结

| 维度 | 覆盖状态 | 主要工具 |
|---|---|---|
| Attendance（频率）| 相对充分（21个工具）| CAPE / PEM-CY |
| Involvement（投入）| 严重不足（10个工具）| PEM-CY 是最完整 |
| Activity Competence | 过度测量（常被误认为参与）| 多数功能评估工具 |
| Preferences（偏好）| 不足 | CAPE/PAC |
| Sense of Self（自我感）| 极少 | 极少工具 |

---

### O6：对参与干预研究的方法论含义

```
如果一项干预研究：
使用了只测量频率的工具（如出席率）
→ 只能声称"改善了出席"
→ 不能声称"改善了参与质量（Involvement）"

如果一项干预研究：
使用了活动能力测量工具（如 BOTMP、粗大运动量表）
→ 只能声称"改善了功能或活动能力"
→ 不能声称"改善了参与"

真正能声称"改善了参与"需要：
同时测量 Attendance + Involvement
使用参与专属工具（如 PEM-CY）
在真实生活情境中测量（不只在治疗室）
```

## Layer 3 · Professional Meaning

**给语言治疗师、特教老师、OT、研究者的启示：**

- **选择测量工具时，区分"测量活动能力"和"测量参与"**：如果干预目标是改善参与，评估工具应该是 PEM-CY 等参与专属工具，而不是运动或功能量表
- **声称"改善了参与"需要有参与测量数据**：用出席频率提升或功能改善来证明参与改善，在方法论上是不严谨的
- **Involvement 是测量最薄弱的维度**：现有工具中测量 Involvement 的极少——这是临床评估中最容易被忽略、但最重要的参与质量信号
- **PEM-CY 是目前最接近 fPRC 框架的测量工具**：在需要同时评估 Attendance + Involvement + Environment 的情况下，PEM-CY 是最优选择

| 研究语言 | ET 方向（Anori Voice）|
|----------|----------------------|
| 118 tools, only 10 measure involvement | 研究者整理了 118 个声称测量"参与"的工具，发现只有 10 个真正把"投入程度"作为独立维度来测量 |
| Most tools = attendance only | 大多数工具测量的是孩子去了多少次，而不是孩子在去的时候有多投入——这是两件完全不同的事 |

| PEM-CY fills the gap | 在这些工具里，PEM-CY 是少数同时覆盖出席频率、投入程度和环境因素的工具 |
