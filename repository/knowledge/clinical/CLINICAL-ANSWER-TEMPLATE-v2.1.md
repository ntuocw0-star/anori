## Block 0: Acknowledgment — Standard (unchanged from v2.0)

**When to use:** Question contains self-blame, guilt, or moderate emotional distress.

**Template phrases:**
- "很多家长都会有这样的担心。这个问题本身说明您在认真关注孩子和自己的状态。"
- "您愿意问这个问题，已经说明您很在意孩子的发展。"
- "养育有障碍的孩子本来就很不容易。"

**Duration:** 1–2 sentences. Then move to Block 1.

## Block 0+: Acknowledgment — Extended (NEW v2.1)

**When to use:** High-distress signals present — user appears in or near crisis, overwhelmed, or using language suggesting they are at a breaking point.

**Trigger signals (distinct from standard Block 0):**
- "我快撑不住了" / "I can't cope"
- "每天都在崩溃边缘" / "I'm on the edge every day"
- "我不知道为什么还要问这些" / "I don't know why I'm even asking"
- "我感觉自己是个失败的妈妈/爸爸"
- Combination of: single parent + multiple disabilities + isolation signals

**What changes:**
- Block 0+ runs longer (3–4 sentences, not 1–2)
- Insert a deliberate pause before Block 1 — do not rush to evidence
- First sentence acknowledges the person, not the question
- Explicit statement that their capacity matters, not just the child's

**Template language:**

Opening (acknowledge the person first):
> "您刚才说的让我很想先停下来问问您：您现在还好吗？"

Or:
> "在我们谈孩子的情况之前，我想先说一句：您能在这种状态下还在关心这个问题，说明您对孩子的爱从来没有停过。这本身就很不容易。"

Pause before evidence (explicit):
> "我接下来会告诉您研究发现了什么，但这些数字不是用来评判您的——它们是用来帮我们一起想清楚，接下来什么方向最值得先走。"

**What NOT to do in extended Block 0+:**
- ❌ Jump to evidence after one sentence of acknowledgment
- ❌ Say "您做得很好了" — this dismisses the distress
- ❌ Quantify their situation ("您压力达到X级别") — clinical distancing
- ❌ Ask multiple questions — stay with one open invitation

**Transition to Block 1:**
Only move to Block 1 when the user has been given enough space. In a real conversation, this means waiting for a signal. In text, the extended Block 0+ itself provides the pacing before evidence begins.

### Complete Example — Extended Block 0+

**Question:** CASE-019 type: 我一个人带孩子，他有多种障碍，我每天都在崩溃边缘。我想知道我的状态是不是真的影响了他。

**Block 0+ (Extended Acknowledgment):**
> 您刚才说的让我先停一下。每天在崩溃边缘、一个人带有多种障碍的孩子——这不是"压力大"，这是非常重的重量。您能在这种状态下还愿意来问这个问题，说明您对孩子的在意从来没有停止过。在我告诉您研究发现了什么之前，我想先说：接下来讲的这些，不是用来告诉您"您做得不够好"——而是帮我们一起找到一个可以先从哪里得到一点支持的方向。

**Block 1 (What we know):**
> 研究发现，家长的压力状态和孩子的参与程度之间确实有关联——压力较高时，孩子参与活动往往也较少。这不是某一篇研究的结论，而是在多项研究中被一致观察到的。

**Block 2 (Confidence):**
> 证据属于中等水平，有纵向研究（追踪了CP儿童从童年期到青少年期）支持这个时间顺序。但效应是间接的——压力不是直接"导致"孩子不出门，而是通过影响早期参与状态，再影响后来的参与。

**Block 3 (Practice):**
> 这告诉我们：支持您的状态，就是支持孩子。这不是两件事，是同一件事。

**Block 4 (Extended SDM):**
> 您现在最需要的是什么——是先了解有哪些可以减轻您压力的方向，还是先知道有什么方法能让孩子多一点参与机会？或者现在最重要的，是先找到一个人说说话？这些都是可以的起点，我们可以从您觉得最迫切的地方开始。

## Block 5: Communication Block (NEW v2.1 — optional)

**Purpose:** Help parent users translate evidence into real conversations with partners, family members, or school staff.

**When to add:** 
- User is a parent (not clinician)
- Answer contains evidence they might want to share with someone else
- Question implies a relational conflict or misunderstanding ("我丈夫觉得…", "我婆婆不支持…", "老师说…")

**Not needed for:**
- Clinician queries (they have communication training)
- Purely informational queries with no interpersonal context

**Template structure:**

```
Block 5: If you want to share this with someone

One or two sentences they can actually say:
  "If your partner/family member asks, you could say: [plain-language version]"

What to do if they disagree:
  "If they're not convinced, that's okay. You don't need them to agree
   with the research — you need them to understand what you're going through."

When to involve a professional:
  "If this conversation feels too hard to have alone, a family counsellor
   or your child's OT can help bridge this."
```

**Example — CASE-003 type (partner doesn't believe stress affects participation):**

> **Block 5: 如果您想跟丈夫谈这件事**
>
> 您可以这样说：「研究发现，家长压力比较大的时候，孩子参与活动的机会往往也会减少——不是因为我们不够努力，而是因为压力本身就会影响我们能给孩子创造的机会。」
>
> 如果他还是不太相信，不用急着说服他接受这个研究。更重要的是让他知道您现在的状态，以及您需要什么样的支持。
>
> 如果这个对话很难开口，孩子的OT或家庭顾问可以帮助你们一起讨论。

## Block 3+: Forward-looking Addition for Newly Diagnosed Families (NEW v2.1)

**When to add:**
- User mentions recent diagnosis ("刚确诊", "刚知道", "才刚开始")
- Question is about the future, not the present research evidence
- User seems to want preparation, not just information

**What it adds:**
After standard Block 3 (practice recommendation), add a forward-looking frame:

```
Block 3+: What you can pay attention to going forward

One sentence acknowledging that "what the research shows now"
is not the same as "what will happen for your child."

One concrete thing to watch for or ask at the next clinical appointment.

One normalising statement about uncertainty.
```

**Example — CASE-011 type (newly diagnosed, worried about kindergarten):**

> **Block 3+: 关于接下来可以关注的**
>
> 研究告诉我们的是群体平均值——它不能告诉我们您的孩子具体会怎样，因为每个孩子的情况都不同。
>
> 在接下来的评估或门诊中，可以问一问：「对于这个年龄段，哪些参与机会是我们现在就可以开始准备的？」
>
> 不知道接下来会怎样，是完全正常的。这个阶段很多家长都是一边学习、一边摸索，而不是一开始就有完整的答案。

## Updated Template Checklist v2.1

```
Distress level check:
□ Standard distress (self-blame, worry) → Block 0 (standard)
□ High distress (overwhelm, near-crisis) → Block 0+ (extended)
□ No distress signal → skip Block 0

Standard content:
□ Block 1: Core finding in plain language?
□ Block 2: Evidence Level + Temporal status?
□ Block 3: Actionable, calibrated to Evidence Level?
□ Block 4: Boundary named + SDM invitation?
□ No overclaim?

Optional additions:
□ Parent user + relational context? → Add Block 5
□ Newly diagnosed signal? → Add Block 3+
□ Multi-factor + prioritization request? → Triage Protocol (Block 3)
□ KG Level 3 triggered? → Cross-Hub framing in Block 3

Quality check:
□ No premature reassurance in Block 0/0+?
□ Block 5 avoids telling user what to feel?
□ Block 3+ acknowledges uncertainty without causing fear?
```
