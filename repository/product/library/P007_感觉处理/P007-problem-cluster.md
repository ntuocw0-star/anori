## Cluster 1 · 日常护理困难

**PRB 标题（家长语言）：** 剪头发、刷牙、洗头，每次都像打仗

**为什么在一起：**
这些都是对头皮、口腔、皮肤的直接接触——触觉高度集中的场景。
触觉敏感的孩子会把这些普通护理体验成真实的不适甚至痛苦，而不是在"故意"。

**涵盖的家长搜索词：**
- 孩子不让剪头发
- 剪头发要按住才行
- 洗头每次哭
- 刷牙是噩梦
- 不让剪指甲
- 理发店进不去
- 不让医生检查
- 洗澡时候哭

**核心误读（家长容易误判为）：**
- 故意不配合
- 脾气差
- 被宠坏了

**Anori 会说：**
这不是脾气，是感觉。他感受到的不适，比你想象的真实。

**PRB ID（待分配）：** PRB-P007-001

## Cluster 2 · 声音世界太吵

**PRB 标题（家长语言）：** 普通的声音，孩子反应像是在受苦

**为什么在一起：**
这些都是突然出现、音量较大、难以预测的声音。
听觉敏感的孩子对这类声音的神经反应更强烈——不是矫情，是神经系统的真实差异。

**涵盖的家长搜索词：**
- 怕吸尘器
- 怕吹风机
- 冲马桶会尖叫
- 怕铃声
- 去商场会崩溃
- 怕气球爆
- 怕雷声
- 餐厅太吵坐不住
- 一有响声就捂耳朵
- 学校铃声每次都很难

**核心误读（家长容易误判为）：**
- 胆子太小
- 听力有问题（去查过没问题）
- 太敏感了，长大就好了

**Anori 会说：**
他的耳朵没有问题。是他的大脑处理声音的方式不同——那个音量，对他来说真的更响。

**PRB ID（待分配）：** PRB-P007-002

## Cluster 3 · 一直想获得感觉

**PRB 标题（家长语言）：** 一直转圈、咬东西、到处摸——停不下来

**为什么在一起：**
这些都是感觉寻求行为（sensory seeking）——孩子的神经系统需要更多输入来感知自己的身体。
转圈提供前庭觉，咬东西提供口腔本体觉，摸东西提供触觉。形式不同，驱动相同。

**涵盖的家长搜索词：**
- 一直转圈转不晕
- 停不下来
- 喜欢跳，一直跳
- 咬衣服
- 咬手
- 把东西放嘴里
- 一直闻东西
- 一直摸别人
- 喜欢蹭人
- 走路要摸墙
- 手总要拿着什么

**核心误读（家长容易误判为）：**
- 多动症（ADHD）
- 坏习惯
- 口欲期没过
- 故意惹人

**Anori 会说：**
这些行为不是调皮，是孩子的身体在自己找它需要的东西。

**PRB ID（待分配）：** PRB-P007-003

## Cluster 4 · 衣服和食物有很多"规矩"

**PRB 标题（家长语言）：** 孩子对衣服和食物有很多"不行"，说不清楚为什么

**为什么在一起：**
衣物和食物都涉及触觉（皮肤感觉和口腔感觉）。
触觉敏感的孩子对材质、接缝、温度、口感的感受更强烈，这些"规矩"背后是真实的不适。

**涵盖的家长搜索词：**
- 袜子穿上就脱
- 袜子接缝要对齐
- 衣服标签必须剪掉
- 只穿旧衣服不穿新的
- 不喜欢特定材质（毛料）
- 挑食很严重
- 只吃三四样东西
- 不吃混在一起的食物
- 不接受颗粒感
- 食物加东西就发现
- 换了品牌不吃

**核心误读（家长容易误判为）：**
- 故意挑剔
- 被宠坏的挑食
- 强迫症
- 性格太执拗

**Anori 会说：**
那件衣服不是"不好看"，是真的让他不舒服。那口食物不是"不好吃"，是他的口腔真的有不同的感受。

**PRB ID（待分配）：** PRB-P007-004

## Cluster 5 · 身体不知道用了多大力

**PRB 标题（家长语言）：** 孩子不知道自己有多大力气，经常弄坏东西或弄伤别人

**为什么在一起：**
本体觉（身体位置感）让我们知道自己在用多大力。
本体觉处理不佳的孩子，需要更用力才能感知自己的身体——所以他们用力太大、撞东西、推人，不是故意的，是感知出了差异。

**涵盖的家长搜索词：**
- 用力太大
- 抱人太用力
- 写字把纸弄破
- 拿东西会摔
- 经常撞到门框
- 经常撞到东西
- 推同学（不是故意的）
- 跟宠物玩太用力
- 不知道轻重
- 开关门很大声
- 走路很响

**核心误读（家长容易误判为）：**
- 不懂事
- 故意欺负别人
- 笨拙
- 协调性不好（只查粗大运动）

**Anori 会说：**
他不是故意的。他感觉不到自己用了多大力——所以他需要用更大的力才能感觉到自己。

**PRB ID（待分配）：** PRB-P007-005

## Cluster 6 · 环境一复杂就崩溃

**PRB 标题（家长语言）：** 孩子在人多、嘈杂、变化多的地方就崩溃，我不知道怎么办

**为什么在一起：**
这是多个感觉系统同时被激活的情境——声音、视觉、触觉、气味、意外变化全部涌入。
当神经系统无法同时处理这些输入，就会崩溃（meltdown）或关机（shutdown）。

**涵盖的家长搜索词：**
- 超市里总会崩溃
- 生日派对哭着跑出来
- 学校餐厅吃不下
- 医院诊所很难配合
- 游泳课换衣服崩溃
- 节假日不想出门
- 人多地方走一会儿就不行
- 突然的变化无法接受
- 从一个活动换到另一个很难
- 学校回来就崩溃

**核心误读（家长容易误判为）：**
- 社交焦虑
- 情绪管理问题
- 任性
- 不适应幼儿园 / 上学

**Anori 会说：**
不是他不想去，也不是他故意制造麻烦。是他的系统在那个环境里真的装不下这么多。

**PRB ID（待分配）：** PRB-P007-006

## P007 的核心写作原则（从 Problem Universe 延续）

> 不说"这是感统问题"，
> 而说"这可能是孩子的神经系统在用行为告诉你它需要什么"。

每个 PRB 的核心工作是：
**帮家长从"这是行为 / 性格 / 管教问题"转向"这是神经系统处理方式的差异"。**
