## 1. Project Summary

**User Path**: P005 — 孩子社交参与
**Core Problem**: 孩子在人群里沉默、无法加入、维持不了关系——家长不知道从哪里帮
**One Sentence**: 社交不是让孩子变成别的样子，而是帮他在这个世界里找到属于他的方式存在。
**Primary Audience**: 有社交参与困难孩子的家长（神经发展差异儿童为主）
**Domain Coverage**: Social Participation, Friendship, Peer Relations

**核心定位**:
不是社交技巧训练，而是帮家长理解社交参与的真实结构——找到孩子的起点，降低代价，在对的环境里建立真实的连结。

**Platform Principle**:
社交代价（Social Cost）是核心概念。
孩子不是不会社交，而是社交对他们需要付出的代价比别人高——找到降低代价的方式，才是真正的支持。


**P005 特有逻辑**:
Support+ 形成一条从「开口」到「建立系统」的完整路径：
- S001: 开口 → 降低开口阻力，等待是最好的支持
- S002: 维持 → 友谊断在「没有下一步」，帮他想那个下一步
- S003: 修复 → 误解不是终点，修复只需要一句话
- S004: 环境 → 对的环境比努力更重要，代价低才能真实出现
- S005: 角色 → 支持不是代替，脚手架让他自己走
- S006: 时间 → 进步是积累，换一个衡量方式

## 5. One Outcome / One Moment 汇总

**路径核心 One Outcome**:
帮助家长从「怎么让孩子变得合群」走到「帮他找到属于他的方式，在这个世界里存在」

**路径核心 One Moment**:
第一次问「他现在站在哪里」，而不是「他什么时候能交到朋友」

**P005 核心概念**:
- 社交代价（Social Cost）：每次社交需要付出的认知和情绪资源
- 社交起点：他现在已经能做到的最小单位
- 渐进阶梯：一次只往前走一步
- 兴趣路径：从兴趣出发，社交在兴趣里自然发生
- 脚手架支持：家长提供框架，孩子完成社交

**品牌翻译语言（全路径）**:
- 不会社交 ≠ 不想社交
- 一个人 ≠ 孤独（主动独处 vs 被迫孤立）
- 社交技巧训练 ≠ 真实连结
- 支持 ≠ 代替
- 进步 ≠ 「交到很多朋友」
- 误解 ≠ 关系结束

## 10. UI Pattern & Design Tokens

**P005 特有设计语言**:
- Social Cost Flow：累加路径图，用 + 号叠加，最后等于「精疲力竭」
- Starting Grid：绿色「已经能做到」标签 + 蓝色「下一步」小字
- Barrier Flow：开口内部流程，最后一行红色「窗口已经关了」
- Wait Box：蓝色 Core Box，「等待是最好的支持」
- Phase Stack：红/蓝/绿三色阶段卡（冷静/理解/修复）
- Role Grid：绿色支持 vs 灰色等等，不用红色×
