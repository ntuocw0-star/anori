## PRB-P006-001 · 我已经撑不住了

**PRB 标题（家长语言）：** 我已经撑不住了，但我不能停下来

**为什么在一起：**
这些都是照顾者倦怠（caregiver burnout）的信号——不是脆弱，是长期承受超出个人能力的压力后的真实反应。

**涵盖搜索词：**
- 我快撑不住了
- 我太累了
- 我感觉被掏空了
- 我不知道还能坚持多久
- 我整个人都垮了
- 我每天都在硬撑
- 我不敢休息

**真实触发场景：**
- 孩子今天又崩溃了，自己也跟着崩溃
- 睡前意识到今天又没有一件事是为自己做的
- 身体出现症状，但没有时间也没有理由停下来

**核心误读（家长容易误判为）：**
- 脆弱
- 不够爱孩子
- 别的家长都做得到，我怎么就不行

**Anori 会说：**
你感到撑不住，不是因为你不够强，而是因为你一直在独自承担太多。这不是性格问题，是处境问题。

**对应 Journey 页面：** journey · s001

## PRB-P006-002 · 没有人理解我

**PRB 标题（家长语言）：** 我身边没有人真的懂，我不知道能跟谁说

**为什么在一起：**
这些都是照顾者孤立（caregiver isolation）——不仅是外部没有支持，也因为"说了别人也不懂"而主动放弃寻求理解。

**涵盖搜索词：**
- 没有人理解我
- 我一个人在面对这一切
- 我不敢说，说了也没用
- 我感觉很孤单
- 身边没有人懂
- 家人不理解
- 朋友不懂

**真实触发场景：**
- 朋友在聊孩子普通成长的烦恼，插不进去
- 试着跟家人说，被说"你想太多了"
- 一个人去参加孩子的评估，回来没有人问今天怎么样

**核心误读（家长容易误判为）：**
- 我太敏感了
- 别人也有难处，我不该抱怨
- 是我自己不会沟通

**Anori 会说：**
你感到没有人理解，不是因为你要求太多，而是因为这段经历本身很难被没有走过的人理解。需要被看见，是正当的。

**对应 Journey 页面：** journey · s002

## PRB-P006-003 · 我不知道该怎么办

**PRB 标题（家长语言）：** 我试了很多，还是不知道接下来该怎么走

**为什么在一起：**
这些都是方向感丧失——不是能力问题，而是在信息过载 + 支持不足 + 没有人整合这一切的情况下，家长会自然陷入不知道下一步的状态。

**涵盖搜索词：**
- 我不知道该怎么办
- 我不知道接下来要做什么
- 我不知道去哪里找帮助
- 我试了很多方法都没用
- 没有人告诉我下一步
- 我感觉迷失了

**真实触发场景：**
- 刚收到新的诊断报告，不知道从哪里开始
- 干预计划太多、太分散，不知道什么最重要
- 专业人员给了建议，但没有人帮助整合成日常能执行的计划

**核心误读（家长容易误判为）：**
- 我不够努力
- 是我没有找到正确的方法
- 别的家长都知道怎么做，只有我不知道

**Anori 会说：**
感觉不知道下一步，不是因为你能力不足，而是因为没有人帮你把这些拼在一起。迷失是合理的，不是失败。

**对应 Journey 页面：** journey · s003

## PRB-P006-004 · 家里没有支持

**PRB 标题（家长语言）：** 家里人不理解，甚至不相信孩子有问题

**为什么在一起：**
这些都是家庭内部分歧——另一半不支持、祖父母否认、家人意见不一，让主要照顾者既要独自承担照顾，还要独自承担"说服别人"的压力。

**涵盖搜索词：**
- 家里人不支持
- 另一半不相信孩子有问题
- 祖父母觉得没事
- 家里意见不一致
- 家人觉得我大惊小怪
- 我一个人在做所有的事
- 没有人帮我分担

**真实触发场景：**
- 和另一半又一次因为孩子的问题吵架
- 祖父母帮忙带孩子，但不执行任何干预建议
- 家人当着孩子的面说"他没事的，你太紧张了"

**核心误读（家长容易误判为）：**
- 是我不会沟通，所以家人不理解
- 是我太焦虑，才让家人觉得没必要担心
- 只要我坚持，他们最终会理解的（等待代替行动）

**Anori 会说：**
家庭内部的分歧，是这条路上最消耗心力的部分之一。独自面对孩子的事，同时还要处理家人的不理解，这份重量值得被正视。

**对应 Journey 页面：** s002 · s004

## PRB-P006-005 · 我一直在责怪自己

**PRB 标题（家长语言）：** 我一直觉得是我的错，不管别人怎么说

**为什么在一起：**
这些都是照顾者的内疚与自责——在没有足够解释的情况下，家长的大脑会自动寻找"是谁的错"，而最容易指向自己。

**涵盖搜索词：**
- 都是我的错
- 是不是我怀孕的时候做错了什么
- 是不是我养育方式有问题
- 我一定哪里做错了
- 我是不是不够好的妈妈/爸爸
- 我有时候对孩子发火，然后很内疚
- 我恨自己
- 我不应该这样

**真实触发场景：**
- 看到文章说某些因素可能影响发育，开始回想自己做过什么
- 发火之后的深夜，反复回想
- 被问到"你们家有没有遗传？"的时候
- 孩子哭闹，自己没有耐心，事后觉得对不起孩子

**核心误读（家长容易误判为）：**
- 内疚说明我是好家长（反向自我安慰）
- 继续自责能让我做得更好（以为内疚有驱动力）
- 别人都说不是我的错，但我就是觉得是

**Anori 会说：**
自责不是你不够好，而是你太在乎了，在没有足够支持的情况下，只能把原因指向自己。这是很常见的，也是值得被认真对待的，而不是被简单安慰"不是你的错"就过去。

**对应 Journey 页面：** journey · s005

## PRB-P006-006 · 我需要照顾自己吗

**PRB 标题（家长语言）：** 我知道应该照顾自己，但我不知道怎么做，也不确定我有这个权利

**为什么在一起：**
这些都是照顾者自我忽视——不是不知道"要照顾自己"，而是把照顾自己视为奢侈、自私或根本没有条件做到的事情。

**涵盖搜索词：**
- 我不敢休息
- 我停下来就会更崩溃
- 怎么照顾自己
- 喘息服务是什么
- 我需要心理咨询吗
- 我去找帮助是不是太夸张了
- 我可以把孩子交给别人吗
- 我有时候需要一个人待着

**真实触发场景：**
- 有机会休息，但无法停止想着孩子
- 有人提议帮忙，但不知道怎么接受
- 自己生病了，但觉得不能停下来
- 第一次想到"我需要帮助"，但不知道这是不是正当的

**核心误读（家长容易误判为）：**
- 照顾自己是自私的
- 等孩子稳定了再照顾自己
- 别的家长都做到了，我没有资格喊累

**Anori 会说：**
照顾自己不是等孩子好了以后的奖励，而是现在就需要的事。不是因为你值得被奖励，而是因为你能持续给孩子的，取决于你自己还剩多少。

**对应 Journey 页面：** s005 · s006
