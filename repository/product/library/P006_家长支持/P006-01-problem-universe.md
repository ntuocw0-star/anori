## 误判表达

家长在说出上面这些话之前，会先这样描述自己：

| 家长说的话 | 背后真正的意思 |
|-----------|-------------|
| "我就是脾气差" | 我已经精力耗尽，承受着超出能力的压力 |
| "我太敏感了" | 我需要支持但没有得到 |
| "没事，都还好" | 我不知道能向谁说 |
| "孩子需要我，我不能倒下" | 我已经把自己的需求完全放到最后 |
| "别的妈妈都做得到，我怎么就不行" | 我在用错误的标准评判自己 |
| "是我自己的选择，我不该抱怨" | 我不允许自己承认自己需要帮助 |
| "我去找心理咨询是不是太夸张了" | 我已经到了需要支持的地步，但不知道这是正常的 |

## 真实触发场景

家长最可能在这些时刻来到 P006：

**日常积累型：**
- 又一次带孩子去评估，被告知又多了一个建议
- 又一次和学校开会，感觉一个人面对所有人
- 又一次睡前崩溃，不知道明天还怎么继续
- 孩子今天又发了很大的情绪，自己也失控了

**关键时刻型：**
- 刚拿到孩子的诊断，不知道接下来怎么办
- 刚和另一半大吵了一架，关于孩子
- 刚被家人说"你想太多了"
- 第一次意识到"这件事不会很快结束"

**孤立触发型：**
- 朋友在聊孩子的普通成长，自己插不进去
- 看到别人家孩子正常发展，突然很难受
- 一个人带孩子去治疗，回程在车里哭
