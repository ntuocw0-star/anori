# 先做——然后 / First–Then Visual Support Card

**RI-ANORI-0001 · v1.0.0**

## 这份资源是什么

"先做——然后"视觉支持卡帮助儿童在两个活动之间建立可预测的顺序，减少过渡期的焦虑与阻力。它是一种视觉支持工具，不是奖励体系，也不是行为管理工具。

## 版本说明

本资源包含两个版本：

- **空白版**（`first-then-card-blank.pdf`）：填写区完全空白，适合家长手写、贴图或贴照片。
- **预填版**（`first-then-card-prefilled.pdf`）：预填 12 个高频家庭活动，打开即可使用。

## 适合谁使用

- 主要受众：家长、照护者
- 辅助受众：教师、早期干预专业人员
- 目标年龄：2–8 岁（具体适用性请根据儿童实际情况判断）

## 如何使用

1. 打印 A4 尺寸（彩色或黑白均可）
2. 在"先"（FIRST）区域填写或指向当前需要完成的活动
3. 在"然后"（THEN）区域填写或指向之后可以做的事情
4. 向儿童展示卡片，语言简洁：「先刷牙，然后玩玩具。」
5. 活动完成后给予正向回应

## 证据基础

本资源基于以下循证框架设计：

- **视觉支持（Visual Supports）**：使用视觉线索减少对口语指令的依赖，提升活动可预测性。
- **结构化教学（Structured Teaching / TEACCH）**：通过物理与视觉结构帮助儿童理解时间顺序。
