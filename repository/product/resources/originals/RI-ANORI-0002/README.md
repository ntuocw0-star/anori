# 每周家庭观察记录 / Weekly Family Observation Sheet

**RI-ANORI-0002 · v1.0.0**

## 这份资源是什么

一张帮助家庭系统记录一周观察的表格。它不是行为评估量表，也不是专业记录工具——它是家长用来整理自己观察、为沟通做准备的日常工具。

## 适合谁使用

- 主要受众：家长、主要照护者
- 使用时机：周末回顾、门诊复诊前、家校沟通前
- 目标年龄：不限（根据家庭实际情况使用）

## 如何使用

1. 打印 A4 表格（黑白即可）
2. 在顶部填写记录的周次与孩子姓名
3. 每天或隔天简短填写五个栏位——不需要每格都填满
4. 周末在「本周亮点」区域写下最值得记住的一件事
5. 门诊或家校沟通前带上这份记录

## 五个观察维度

| 栏位 | 记录什么 |
|------|---------|
| 参与 / Joined | 孩子参与了什么活动或场景 |
| 亮点 / Highlight | 今天开心或成功的事情 |
| 困难 / Challenge | 遇到的困难（可选填）|
| 观察 / Noticed | 任何值得记录的细节 |
| 分享 / To share | 想和老师或医生说的话 |

## 设计原则

- 正向记录优先：亮点栏优先于困难栏
- 不强制每格填写：空格不是失败
- 没有评分、没有勾选框：只有开放填写空间
