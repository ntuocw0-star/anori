## Anori Voice 是什么

Anori Voice 是一种特定的语言风格，它的工作是：

> 把研究语言转译成家长真正能听进去的话。

不是安慰，不是简化，而是**基于证据的归因转移**——把错误归因修正为更符合证据的归因。

---

## 核心原则

**1. 从行为描述进入，不从诊断进入**
✅ "孩子怕吸尘器"
❌ "孩子有听觉过敏"

**2. 把行为解释为需求，不解释为问题**
✅ "他的身体在寻找它需要的感觉"
❌ "他有感觉寻求行为"

**3. 归因指向处境或神经系统，不指向性格或管教**
✅ "这是他大脑处理信号的方式，不是性格问题"
❌ "他太敏感了" / "是管教出了问题"

**4. 不承诺，不过度安慰**
✅ "这段路是真的难"
❌ "一定会好的" / "你已经做得很好了"

**5. 行动提示是今天可以做的一件具体的事**
✅ "下次见 OT 时，把孩子最怕的三种声音写下来带去"
❌ "积极寻求专业支持"

## 高频叙事模式

### "很多家长后来才发现……"
**使用场景：** ET 层、KA 的 Parent Meaning 区块
**功能：** 建立"不只是你"的陪伴感，同时引入归因转移

**示例：**
- 很多家长后来才发现，真正累的不是 AAC，而是突然要扮演的所有角色。
- 很多家长后来才发现，最大的阻力有时不是来自学校，而是来自家里另一半。

### "不是……而是……"
**使用场景：** reframe 区块、ET 转译、PRB 的误判澄清
**功能：** 直接完成归因转移，句式简洁有力

**示例：**
- 不是孩子不配合，而是触觉让他真的感到不舒服。
- 不是家长不够努力，而是支持系统在关键时刻缺席。
- 不是 AAC 在等他，而是 AAC 在帮他进入语言。

### "这不是……这是……"
**使用场景：** 误解澄清、Path 的 hero_companion
**功能：** 直接替换误判标签

**示例：**
- 这不是脾气，这是感觉。
- 这不是放弃语言，这是通向语言的另一条路。
- 这不是管教失败，这是系统支撑不够。

## 禁用词汇（内容层）

| 禁用 | 替代方向 |
|------|---------|
| 感统失调 / SPD / 触觉防御 | 感觉处理方式不同 / 触觉对他来说更强烈 |
| 照顾者倦怠（burnout）| 撑不住 / 精力耗尽 / 系统超载 |
| 干预 / 治疗 | 支持 / 帮助 / 一起来 |
| 障碍 / 残障 | 差异 / 不同的处理方式 |
| 一定会好的 | 不作承诺，只提供理解和行动 |
| 你已经做得很好了 | 不作空泛安慰，只承认真实处境 |
